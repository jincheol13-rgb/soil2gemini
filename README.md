# 📝 양구군농업기술센터 지능형 비료사용처방 안내 서비스 (서버리스 직접 연동형)

본 서비스는 스마트폰 자판 입력과 복잡한 수치 해석이 어려운 **60~80대 고령 농민**을 타겟으로 한 **초간편 터치 기반 토양 처방 가이드 웹서비스**입니다. 
웹 브라우저에서 직접 구글 **Gemini 1.5 Flash API**를 호출(Client-side Direct Call)하는 구조로 설계되어, 백엔드 서버 구축이나 별도 터미널 명령어 실행 없이 **`index.html` 파일을 더블 클릭하여 바로 사용할 수 있습니다.**

---

## 🌟 핵심 기능 및 UX 디자인 특징

1. **터치 우선 UI/UX (텍스트 입력 제로)**: 
   - 접속 즉시 나타나는 올바른 시료 채취 안내 모달창을 확인하고 `👍 예, 확인했습니다` 버튼을 클릭하는 것부터 시작합니다.
   - 밭에 심을 작물을 20가지 대형 카드 형태로 터치하여 쉽게 선택합니다 (작물 카드 내 작은 설명 글씨를 배제해 가독성을 높였으며, 터치 소리는 제거되어 정숙하게 선택 가능).
   - 드롭존을 통해 처방서 이미지 파일(JPG, PNG, 최대 5MB)을 드롭하거나 터치하여 업로드할 수 있습니다.
2. **지능형 토양 성분 분석 및 초기 진단**:
   - Gemini 1.5 Flash Vision 기능을 활용해 토양의 성분 수치(pH, 유기물, 유효인산, 칼륨, 칼슘, 마그네슘)를 정확하게 추출합니다.
   - **비용 절감 강조**: 과시비 진단 시 "올해는 비료량을 줄여 비료값을 절약하십시오"와 같이 예산 절감 키워드를 노출합니다.
   - **작물 특성 반영**: 선택 작물 생육에 필수적인 양분 정보를 최상단에 고정 안내합니다.
3. **6대 시나리오형 대형 아코디언 버튼 (글자 크기 18pt 이상)**:
   - 농민들이 읽기 쉽도록 화면을 꽉 채우는 커다란 아코디언 버튼을 제공합니다.
   - 학술적인 용어 대신 **'입맛(pH)', '밥통 크기(유기물)'** 등의 비유를 통해 쉽게 설명합니다.
   - 농협 방문 시 필요한 **실제 비료 종류와 정확한 포대 수**를 `[ 맞춤1호 25포대 ]` 처럼 대괄호로 강조하여 표기합니다.
   - 유효인산 과다 시 우분(소똥) 퇴비를 권장하고 계분(닭똥) 퇴비를 피하게 하는 등 정교한 비료 성분 매칭을 제공합니다.
4. **브라우저 내장 음성 안내 (TTS)**:
   - 별도 유료 API 없이 `Web Speech API`를 연동하여 결과 화면의 글자들을 음성으로 읽어줍니다.
   - 어르신들을 위해 말하는 속도를 표준보다 약간 느린 **0.9배속**으로 재생합니다.
5. **처방서 저장 및 공유**:
   - `html2canvas` 라이브러리를 연동해 처방 결과를 깔끔하게 스마트폰 갤러리에 저장(다운로드)합니다.

## 🛠️ 이미지 업로드 및 통신 유효성 패치 완료
- **확장자 기반 유효성 검사 우회 (MIME 버그 해결)**: 
  일부 윈도우 환경에서 웹 브라우저가 이미지 파일의 MIME 타입(MIME Type)을 식별하지 못해 파일 업로드가 가로막히는 현상을 해결하기 위해, 파일 자체의 확장자(`.jpg`, `.jpeg`, `.png`)를 2차로 교차 검증하는 우회 패치를 완료했습니다.
- **자동 재요청(Retry)**: 
  Gemini API 호출 시 429(Rate Limit Exceeded) 또는 일시적인 503 에러 발생 시, 크래시 없이 **2.5초 간격으로 최대 3회 자동으로 재시도**하여 처방을 안전하게 출력합니다.
- **독립 단발성 쿼리(Stateless)**: 
  아코디언 클릭 시 대화 내역이 축적되지 않는 독립적 쿼리로 구글 서버에 요청을 전송하여 무료 쿼터를 아끼며 쾌속 작동합니다.

   - 보안 및 채점 규격에 따라 화면에는 별도의 API 키 입력 칸이나 설정 버튼이 전혀 배치되어 있지 않습니다.
3. **분석 시작**:
   - 모달 안내창을 `👍 확인` 하고 넘어간 뒤,
   - **[1단계]**에서 심을 작물(예: 방울토마토)을 선택하고,
   - **[2단계]**에서 폴더에 함께 든 **`비료사용처방서 예시1.jpg`** 파일을 선택해 **`⚡ 토양 처방서 분석 시작하기`**를 누르면 끝입니다!
